🗳 ECI Online Voting System

Java Web Application using JSP, Servlets & SQL Server

📌 Project Overview

ECI Online Voting System is a secure web application developed using Java, JSP, Servlets and SQL Server. It allows voters to register, log in securely, cast votes online and view election results. The system also provides an Admin Panel for managing candidates, voters and election records.

✨ Main Features

  • 🧑 Voter Registration
  • 🔐 Secure Login System
  • 🗳 Online Vote Casting
  • 👨‍💼 Admin Dashboard
  • 👤 Candidate Management
  • 📋 Voter Management
  • 📊 Live Result System
  • 🔒 One Person One Vote
  • 📄 Election Reports
  • 🚪 Logout System

💻 Technologies Used

Java JSP Servlet JDBC SQL Server HTML CSS JavaScript Apache Tomcat

🎯 Project Objective

The objective of this project is to provide a secure, transparent and efficient online voting platform that simplifies election management while ensuring data security and preventing duplicate voting.
