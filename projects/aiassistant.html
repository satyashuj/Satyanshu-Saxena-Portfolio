🤖 Swing AI Assistant

Java Swing Desktop Application

📌 Project Overview

Swing AI Assistant is an intelligent desktop application developed using Java Swing. It combines Artificial Intelligence with desktop technologies to help users interact through text, voice and documents.

🚀 Features

  • 🤖 AI Chat Assistant
  • 🎤 Voice Input (Speech to Text)
  • 🔊 Text to Speech
  • 📄 PDF Reader & AI Summary
  • 🖼 OCR (Image to Text)
  • 📁 File Upload & Management
  • 🌐 Multi Language Support
  • 💬 Chat History
  • 📥 Export Chat as PDF & TXT
  • 🌙 Dark / Light Mode
  • 📸 Camera Capture
  • 🔍 AI Image Analysis

💻 Technologies Used

Java Swing Ms-SQL PDFBox Tess4J OCR Apache POI Groq AI Java Sound API

🎯 Objective

The objective of this project is to provide a smart AI-powered desktop assistant capable of answering questions, reading PDFs, recognizing text from images, processing voice commands and improving productivity through Artificial Intelligence.
