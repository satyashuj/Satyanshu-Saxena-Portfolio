🏙 Smart City Portal

Java Web Application for Smart City Services

📌 Project Overview

Smart City Portal is a web-based citizen service platform developed using Java, JSP, Servlets, JDBC and MySQL. It provides online government services, complaint management, bill payment and scheme information through a modern user-friendly interface.

✨ Key Features

  • 📝 Online Complaint Registration
  • 📊 Complaint Tracking System
  • 💡 Electricity Bill Payment
  • 🚰 Water Bill Payment
  • 🏛 Government Scheme Information
  • 📢 City News & Announcements
  • 👤 Citizen Registration & Login
  • 👨‍💼 Admin Dashboard
  • 📂 Complaint Management
  • 📈 Reports & Analytics

💻 Technologies Used

Java JSP Servlet JDBC MySQL HTML CSS JavaScript Bootstrap Apache Tomcat

🎯 Project Objective

The objective of this project is to digitize citizen services by providing a single online platform where users can access government services, register complaints, pay utility bills and stay informed about public schemes efficiently.
