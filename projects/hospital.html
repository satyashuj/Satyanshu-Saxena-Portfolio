🏥 Hospital Management System

Java Swing Desktop Application

📌 Project Overview

Hospital Management System is a Java Swing desktop application designed to manage daily hospital operations. It provides an easy-to-use interface for handling patient records, doctor information, appointments, billing, prescriptions and medical reports.

✨ Key Features

  • 👨‍⚕️ Doctor Management
  • 🧑‍🤝‍🧑 Patient Registration
  • 📅 Appointment Scheduling
  • 💊 Prescription Management
  • 🛏️ Ward & Room Management
  • 🧾 Billing & Payment System
  • 📋 Medical History Records
  • 🔍 Search Patient Details
  • 📈 Hospital Reports
  • 🔐 Admin Login

💻 Technologies Used

Java Java Swing JDBC Ms-SQL NetBeans OOP AWT

🗄 Database Tables

  • Patients
  • Doctors
  • Appointments
  • Departments
  • Billing
  • Medicines
  • Users

🎯 Project Objective

The objective of this project is to automate hospital operations by providing a secure desktop application that efficiently manages patients, doctors, appointments, billing and hospital records, reducing paperwork and improving healthcare services.
